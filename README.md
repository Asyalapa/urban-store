# UrbanStore

Интернет-магазин товаров для дома: свечи, растения, декор, ароматы, текстиль и освещение.

Стек: vanilla JS (ES modules), HTML, CSS.
**[🔗 Посмотреть сайт](https://asyalapa.github.io/urban-store/)**

## Статус

Проект на стадии разработки. Реализован базовый функционал:
- каталог с фильтрацией по категориям
- поиск и сортировка
- корзина (localStorage)
- избранное (localStorage)

## Запуск

Просто открой `index.html` в браузере. Сервер не нужен — всё на клиенте.
